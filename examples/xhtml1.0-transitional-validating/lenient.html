In this document we show how to use the LAML mirror rules in a relaxed way

Passing numbers, such as 1, 2, and 3 to the paragraph mirror function.

Yet another example:

Passing characters such as a and b to the paragraph mirror function.

Using non-string values as attribute values.

The procedures set-xml-accept-only-string-valued-attributes-in and set-xml-accept-extended-contents-in are useful to operate LAML in a relaxed mode.
