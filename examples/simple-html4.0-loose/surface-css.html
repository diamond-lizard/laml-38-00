  1. As mentioned, the font-famlily attribute on the body
  2. Red emphasis using the css color attribute
  3. Yellow background color of emphasis
  4. Large letter spacing in emphasis
  5. Sub and superscript
  6. Text transformation to uppper case
  7. Ordered list with lower roman numbers
  8. links with bacground color and no underlining
  9. Borders
