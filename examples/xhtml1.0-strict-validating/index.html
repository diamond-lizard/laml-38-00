Overview of XHTML1.0 examples - with validation

The examples in this directory introduces the XHTML1.0 strict, validating mirror. Notice that this mirror uses the XML-in-LAML framework.

The following simple examples exist in this directory:

Example LAML HTML
Hello world hello-world.laml hello-world.html
Hello world - non valid hello-world-non-valid.laml hello-world-non-valid.html
A simple page simple-1.laml simple-1.html
Another simple page simple-2.laml simple-2.html
A page with tables tables.laml tables.html

There are additional XHTML examples:

  • Examples for the PLAN-X paper
  • Tutorial HTML mirror examples

If your browser will not present files with 'laml' or 'scm' extensions you should disable any special handling of these extensions on your system. In that way they are treated as pure text files.
