Overview of HTML4.01 convenience examples - with validation

The examples in this directory use the convenience library. This library contains a collection of loosely related functions on top of the validting basic and surface libraries.

Some of these you may find useful. Others are probably only important for myself. It is likely that you will develop your own collection of "convenience functions" if you start using LAML.

The following simple examples exist in this directory:

Example LAML HTML

Box, frame, and multi column examples test19.laml test19.html
mail-link etc - simple test18.laml test18.html

If your browser will not present files with 'laml' or 'scm' extensions you should disable any special handling of these extensions on your system. In that way they are treated as pure text files.
