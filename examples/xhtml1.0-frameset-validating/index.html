Overview of XHTML1.0 frameset examples - with validation

The examples in this directory introduces the XHTML1.0 frameset, validating mirror. Notice that this mirror uses the XML-in-LAML framework.

In the examples frameset-2 and frameset-3 we illustrate the coexistence of XHTML1.0 frameset and XHTML1.0 transitional.

To get the full outcome of the examples, you should try processing them in the directory examples/xhtml1.0-frameset-validating/.

The following simple examples exist in this directory:

Example LAML HTML
A very simple frameset frameset-1.laml frameset-1.html
A frameset with language overlap frameset-2.laml frameset-2.html
A frameset without language overlap frameset-3.laml frameset-3.html

If your browser will not present files with 'laml' or 'scm' extensions you should disable any special handling of these extensions on your system. In that way they are treated as pure text files.
