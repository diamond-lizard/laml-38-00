LAML brings HTML and XML to the Scheme programming language

LAML is software package that supports authoring of HTML and XML documents in Scheme Syntax.

The concept LAML: LAML is a Lisp Abstracted Markup LanguageLAML consists of HTML mirrors, document styles, and tools

  • LAML Characteristics

    • Supports a number of different HTML mirrors

      • LAML 4.0 loose. XHTML1.0 strict, transitional, and frameset

    • Supports a number of document styles

      • Domain specific WEB languages

      • LENO, Scheme manual pages, questionnaire, ...

We here mention a number of important LAML characteristics
