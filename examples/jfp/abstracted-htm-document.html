WEB Programming in Scheme - the LAML approach

The paper WEB Programming in Scheme - the LAML approach is written for people who are interested in functional programming. The abstract is available as a separate page.

The paper contains the following sections:
  1. Introduction
  2. Markup language mirroring
    1. Basic Mirroring
    2. Discussion of the Mirror
    3. The creation and organization of the mirror
  3. Programming with the HTML mirror functions
  4. Raising the level of abstraction
  5. LAML overview
  6. Similar
  7. Conclusions

There exists three other papers about LAML by the same author:

  • Programmatic WWW authoring using Scheme and LAML
  • Using Lisp as a markup language---The LAML approach
  • Programming World Wide Web Pages in Scheme

The home page of the LAML project contains additional information, including downloading of LAML as free software.
