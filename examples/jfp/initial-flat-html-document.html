WEB Programming in Scheme - the LAML approach

The paper WEB Programming in Scheme - the LAML approach is written for people who are interested in functional programming. The abstract is available as a separate page.

The paper contains the following sections:

  1. Introduction
  2. Markup language mirroring
  3. Programming with the HTML mirror functions
  4. Raising the level of abstraction
  5. LAML overview
  6. Reflections and similar work
  7. Conclusions
