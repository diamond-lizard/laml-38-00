Manual page examples

The following manual examples exists in this directory:

ExampleLAMLHTML
A very simple manual pagemanual-page-1.lamlmanual-page-1.html

There exists relevant templates of the entire manual page (M-x insert-laml-template) and of sections (M-x insert-manual-section) and pages (M-x insert-manual-page) such that it is very easy and safe to construct the example page above from Emacs.

In addition, the source files of several manual pages in this distribution are available in the directory laml/lib/man/ . By studying these you can see how the Schemedoc tool can be used together with the manual style.

If your browser will not present files with 'laml' or 'scm' extensions you should disable any special handling of these extensions on your system. In that way they are treated as pure text files.
