Bibtex page examples

The following bibtex examples exists in this directory:

Example Bibtex Parsed Lisp file HTML
A few bibtex entries sample.bib sample.lsp sample.html

The sample.lsp file, which represents the parsed file, has been pretty printed in order to make it more readable. It is a list of association lists.

The sample.html file represents the HTML pretty printing of the sample bibtex file, using the presentation functions of the LAML bibtex tool. The index.laml file, which creates this page, activates the LAML bibtex procedure, and it pretty prints the resulting sample.lsp file, in which the association list representation resides.
