Elucidator Download and Installation

The Scheme Elucidator is located in the styles/elucidator directory of the LAML software distribution.

The LAML software distribution can be downloaded from the LAML download page:

  • LAML distribution

Follow these steps in order to install and set up the Scheme Elucidator:

  1. Set up LAML - see LAML setup instructions . You should be sure also to setup the LAML Emacs support, as described in these setup instructions.

  2. Try out the elucidator on the example provided in the directory: examples/elucidator/ of your LAML system.

    1. Process the laml file in the directory with C-o from Emacs

    2. Set up the Emacs editor support for this particular elucidative program with M-x setup-elucidator from within Emacs

From Emacs activate M-x elucidator-help. See also the manual of the Scheme elucidator .
