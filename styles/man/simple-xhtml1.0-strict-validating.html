LAML Source file: styles/simple-xhtml1.0-strict-validating.scm

This is the XML-in-LAML mirror of XHTML1.0-strict which is derived from the XHTML DTD.

The XHTML 1.0 mirror functions and the generic XML-in-LAML library are both important for this style.

The use of this library takes care of loading a number of useful libraries (time, color, file-read, and of course the XHTML1.0 mirror libraries mentioned above). The use of this document style makes life a little easier for you, compared to a manual loading of the mentioned LAML libraries.

Usage: (laml-style "simple-xhtml1.0-strict-validating")

There are no external definitions in this library.
