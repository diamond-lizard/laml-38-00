LAML Version 26.0 (January 18, 2005, alpha-3)

This is the XML-in-LAML mirror of XHTML1.1 which is derived from flattened the XHTML 1.1 DTD. Document validation is integrated in the XHTML1.1 mirror functions.

The XHTML 1.0 mirror functions and the generic XML-in-LAML library are both important for this style.

The use of this library takes care of loading a number of useful libraries (time, color, file-read, and of course the XHTML1.1 mirror library mentioned above). The use of this document style makes life a little easier for you, compared to a manual loading of the mentioned LAML libraries.

Usage: (laml-style "simple-xhtml1.1-validating")

There are no external definitions in this library.
