Historial notes: LAML with MzScheme on older Windows Systems

Earlier on I used MzScheme on Windows 98 and NT4.0, relatively problem free.

It did not like Windows 98, because processes were hard to kill.

NT4.0 was OK, although the laml command prompt did not work well.

The last couple of years I have not tried LAML on Windows 98 nor NT. I use mostly Unix, including Linux, Windows 2000, and Windows XP to some degree.
