LAML with MzScheme 101-103 on Windows

If you already have an old PLT version (101, 103) on your Windows machine you can use it with LAML.

If not, you should download the newest version MzScheme, or the newest version of DrScheme (which includes MzScheme).

I have used MzScheme 201, 202, 203, and 204 extensively on Windows 2000 with LAML. I keep a version 103 present, and I use it every now and then, problem free.
