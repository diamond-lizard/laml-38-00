Coexisting LAML distributions

You may have downloaded both SchemeDoc and one of the more comprehensive LAML distributions.

The question is now if they can co-exist.

The major problem is that the Emacs stuff, as loaded from your .emacs file (via loading of the file emacs-support/laml-emacs-support.el ) collide if you load Emacs support from more than one distribution simultaneously.

Therefore you should make sure, that only one laml-emacs-support.el file is loaded in your .emacs file. This can be done by some simple Emacs Lisp programming.
